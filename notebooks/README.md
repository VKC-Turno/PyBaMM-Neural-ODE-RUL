# Notebooks

Start here:
- `notebooks/01_project_overview.ipynb` — end-to-end visibility into what the repo currently does

Launch (recommended):
```bash
cd /home/hj/Desktop/PINNs
.venv/bin/python -m pip install notebook
.venv/bin/python -m notebook
```

Tip: if you want the notebook to use the project venv kernel, install:
```bash
.venv/bin/python -m pip install ipykernel
.venv/bin/python -m ipykernel install --user --name pinns-venv --display-name "PINNs (.venv)"
```
